# sharks-client

This project is generated with [yo angular generator](https://github.com/yeoman/generator-angular)
version 0.11.1.

## Build & development

Run `grunt` for building and `grunt serve` for preview. Do note that http://0.0.0.0:9000 often does not work in your browser, go instead to http://localhost:9000 , that may work better!

## Testing

Running `grunt test` will run the unit tests with karma.
